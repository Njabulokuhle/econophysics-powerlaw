# Econophysics Projects

This folder contains my econophysics simulation projects from my CS + Physics degree.

## Projects

### 1. Power Laws in Stock Returns (`fat_tails.py`)
- Analyzes S&P 500 daily returns (2014-2024)
- Demonstrates that stock returns follow a power law (α = 2.71), not a normal distribution
- Shows that extreme crashes are 100x more likely than Gaussian models predict

### 2. Wealth Distribution Simulator (`wealth_simulation.py`)
- Simulates 1,000 people trading 1% of their wealth randomly
- Shows how fair, random transactions inevitably produce extreme inequality
- Demonstrates the emergence of power law wealth distribution

## How to Run
```bash
python fat_tails.py
python wealth_simulation.py
